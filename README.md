# finder
 
## Installation

Use the package manager [yarn](https://classic.yarnpkg.com/lang/en/docs/install/#windows-stable)/[npm](https://docs.npmjs.com/downloading-and-installing-node-js-and-npm) to install.

```bash
yarn install
```

```bash
npm install
```

## Usage

```bash
yarn run dev
```

```bash
npm run dev
```
